# Hexlet Chat

[![Actions Status](https://github.com/zheshek/frontend-project-12/actions/workflows/hexlet-check.yml/badge.svg)](https://github.com/zheshek/frontend-project-12/actions)

Hexlet Chat — это real-time чат, построенный на современном стеке технологий. Проект позволяет пользователям общаться в реальном времени, создавать каналы и управлять ими.

## 🚀 Демо

https://frontend-project-12-ov12.onrender.com

## ✨ Возможности

- 💬 Общение в реальном времени через WebSocket
- 🔐 Аутентификация и регистрация пользователей
- 📱 Адаптивный дизайн для мобильных устройств
- 🎨 Современный интерфейс на React Bootstrap
- ⚡ Быстрая и отзывчивая работа
- 🔄 Автоматическое обновление сообщений
- 🚫 Фильтрация нецензурных слов
- 👥 Поддержка нескольких пользователей

## 🛠 Технологии

### Frontend
- **React 18** - библиотека для пользовательских интерфейсов
- **Redux Toolkit** - управление состоянием
- **React Bootstrap** - компоненты интерфейса
- **Formik + Yup** - управление формами и валидация
- **Socket.IO Client** - real-time соединение
- **i18next** - интернационализация
- **React Toastify** - уведомления
- **Rollbar** - отслеживание ошибок

### Backend
- **Express** - веб-фреймворк
- **Socket.IO** - real-time соединения
- **PostgreSQL** - база данных
- **JWT** - аутентификация

### Инструменты
- **Vite** - сборка проекта
- **ESLint** - проверка кода
- **Prettier** - форматирование кода
- **Playwright** - e2e тестирование
- **GitHub Actions** - CI/CD

## 📦 Установка и запуск

### Требования
- Node.js 16 или выше
- npm 8 или выше

### Локальный запуск

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/zheshek/frontend-project-12.git
cd frontend-project-12
```

2. **Установите зависимости**
```bash
make install
```

3. **Запустите проект в режиме разработки**
```bash
make dev
```

4. **Откройте браузер** и перейдите по адресу `http://localhost:5001`

### Сборка для production

```bash
make build
make start
```

## 👥 Использование

### Регистрация и вход
1. Перейдите на главную страницу
2. Нажмите "Регистрация"
3. Введите имя пользователя (от 3 до 20 символов) и пароль (не менее 6 символов)
4. После регистрации вы автоматически войдете в систему

### Общение в чате
- Выберите канал из списка слева
- Напишите сообщение в поле ввода
- Нажмите Enter или кнопку "Отправить"

### Управление каналами
- **Создать канал**: нажмите кнопку "+" слева от списка каналов
- **Переименовать канал**: нажмите на троеточие рядом с названием канала
- **Удалить канал**: нажмите на троеточие и выберите "Удалить"
